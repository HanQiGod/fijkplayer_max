**fijkplayer_max** is a media player plugin for flutter ecosystem which wraps the native [ijkplayer](https://github.com/Bilibili/ijkplayer) into flutter and supports both Android and iOS.
**fijkplayer_max** uses [ijkplayer](https://github.com/befovy/ijkplayer) as the player kernel. The ijkplayer uses ffmpeg for audio and video demuxing and decoding, while adding hardware-accelerated decoding capabilities specific to Android and iOS platforms.

**fijkplayer_max** has the same playback capabilities as ijkplayer and is very convenient to use.
**fijkplayer_max's** API is easy to use, and it's easy to add fijkplayer_max to your flutter app. There is no need bu build ffmpeg and native ijkplayer yourself.

fijkplayer_max brings the video from native to Flutter through `Texture`, which has better performance than `PlatformView`. fijkplayer_max has build-in ijkplayer, which hosted separately on jcenter and CocoaPods.

This site contains a lot of fijkplayer_max documentation to help you use and understand fijkplayer_max.
